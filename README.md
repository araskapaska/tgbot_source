TGBOT SOLAR API
Плагин для Solar2D, чтобы создавать Telegram-ботов: отправка сообщений, медиа, inline-кнопки, polling и вебхуки. Создан @xqss_DEVELOPER.
Установка

Скопируй telegram_bot.lua в корень проекта.
